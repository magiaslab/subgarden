# SUBGarden

Sito web e ecosistema di landing page per SUBGarden - Sistema di subirrigazione di lusso per il mercato toscano.

## Stack Tecnologico

- **Next.js 15** (App Router)
- **TypeScript** (Strict mode)
- **Tailwind CSS**
- **Framer Motion**
- **next-intl** (i18n: it, en, de)

## Setup

```bash
npm install
npm run dev
```

Apri [http://localhost:3000](http://localhost:3000) nel browser.

## Struttura Progetto

- `/app` - App Router pages e layouts
- `/components` - Componenti React riutilizzabili
- `/lib` - Utilities e configurazioni
- `/public` - Asset statici

## Lingue Supportate

- Italiano (it) - default
- Inglese (en)
- Tedesco (de)
